San Francisco City Streaming

Mobile / GPUI PRO / point cloud optimization / cell-based streaming

Production-пайплайн для рендеринга крупной городской сцены на основе point cloud / instancing dataset. Переработал тяжёлый монолитный набор данных в cell-based streaming систему с runtime-загрузкой, batch upload в GPUI и контролируемой выгрузкой неактивных клеток.

1.5 GB → 300 MB сжатие исходного dataset
~9 → 60 FPS прирост runtime performance
-90% нагрузка на CPU/GPU
  • Архитектура: спроектировал city streaming runtime вокруг клеток, runtime grid lookup, desired-cell planning, state tracking и контролируемой выгрузки.
  • Data pipeline: реализовал конвертацию из CSV source data в binary chunks, per-cell metadata, runtime catalog и GPUI-ready upload batches.
  • Runtime streaming: загружал только нужные участки города вокруг игрока и выгружал неактивные клетки вместо удержания всего city dataset активным.
  • Optimization: снизил объём runtime-данных, batch overhead и стоимость рендера, устранив перегрев на мобильном устройстве.

Стек: Unity, Mobile, Addressables, GPUI PRO, binary data pipeline, runtime streaming, batching, object pooling.
